Projet réalisé par GUARDIA Quentin (qguardia66@gmail.com)

Pour compiler:
	make
	
Pour lancer le jeu:
	./puissance4
	
Pour supprimer les fichiers objet et l'exécutable:
	make clean
	
Une fois l'exécutable lancé, suivre les instructions pour jouer, lancer les tests unitaires ou bien les simulations.

Structure du dossier:
Dossier data_simul: informations sur les parties simulables dans des fichiers texte
Dossier data_tst: grilles prédéfinies pour les tests unitaires
Dossier doc: rapport écrit au format pdf
Dossier obj: fichiers objet créés lors de la compilation
Dossier src: sources du programme
Fichier makfile: commandes pour compiler le programme
